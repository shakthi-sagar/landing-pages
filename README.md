# landing-pages

Monorepo for product landing pages.

## Apps
- `apps/quickpeek`
- `apps/swiftgpt`
- `apps/lockin`
- `apps/toolbento`

## Setup
- `npm install`

## Dev
- `npm run dev:quickpeek`
- `npm run dev:swiftgpt`
- `npm run dev:lockin`
- `npm run dev:toolbento`

## Build
- `npm run build:all`
